# 投影看板計畫 (Projection Signboard Project)

本計畫旨在建立一個輕量化、模組化的網頁看板系統，專為電視投影顯示設計，具備高穩定性與遠端維修能力。

## 核心價值
- **即時看板**：30 秒自動刷新，四階段流程管理。
- **維護省心**：支援遠端健檢與雲端備援，降低維護門檻。

## 商業服務分級 (Commercial Tiers)

| 服務等級 | 核心價值 | 服務內容 |
| :--- | :--- | :--- |
| **基礎版** (0 元) | 滿足基本看板需求 | 本地看板顯示、年度日誌存儲 |
| **安心維護版** (1,000 元/年) | 數據保險與技術支撐 | **雲端備份** + **遠端健檢** + **年度二次微調** |
| **專業加值版** (另議) | **移動辦公與遠端管理** | 包含以上所有，加開 **Tailscale 遠端存取權限** |

## 技術架構
- **後端**：Python Flask + SQLite。
- **部署**：Docker / Docker-Compose。
- **連線**：使用 Tailscale 進行安全遠端管理與加值存取。

## 硬體災難復原
1. 更換新 SD 卡並燒錄通用映像檔。
2. 啟動後執行 `/app/scripts/sync_data.sh` 即可恢復數據。
