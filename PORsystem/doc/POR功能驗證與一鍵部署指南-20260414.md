# POR 系統 Docker 部署指南 (Windows)

本計畫已準備好，您可以直接在裝有 Docker 的 Windows 電腦上部署 POR 系統。

## 部署環境需求
- 已安裝並啟動 **Docker Desktop**。
- 專案目錄：`d:\PythonWorking\POR\PORsystem`

## 部署步驟

### 1. 開啟終端機
請開啟連結至該目錄的 **PowerShell** 或 **命令提示字元 (CMD)**。

### 2. 執行一鍵啟動
在資料夾中直接雙擊執行：
- **deploy.bat** (高相容性啟動腳本)

### 3. 確認運行狀態
執行以下指令確認容器是否正常啟動：
```powershell
docker ps
```
如果看到 `signboard_app` 狀態為 `Up`，代表部署成功。

### 4. 存取系統
開啟瀏覽器並輸入以下網址：
[http://localhost:5000](http://localhost:5000)

---

## 常見指令備忘

- **停止並移除容器**：`docker-compose down`
- **查看即時日誌**：`docker-compose logs -f`
- **重新啟動系統**：`docker-compose restart`

## IP 顯示與手機存取設定 (Docker 環境必看)

由於 Docker 容器內部的 IP 與您 Windows 電腦的 IP 不同，系統預設抓取的 QR Code URL 可能會失效。請按照以下步驟修正：

### 1. 查詢您的電腦 IP
在 Windows 的 PowerShell 執行：
```powershell
ipconfig
```
找到「無線區域網路」或「乙太網路」下的 **IPv4 位址** (通常是 `192.168.x.x`)。

### 2. 修改設定檔
開啟 [config.inf](file:///d:/PythonWorking/POR/PORsystem/config.inf)，在 `[System]` 區塊填入該 IP：
```ini
manual_ip = 192.168.1.xxx  (請改為您的實際 IP)
```

### 3. 重啟 Docker
修改完成後，請重新執行一遍啟動腳本或執行：
```powershell
docker-compose restart
```

---

## 手機存取驗證
1. 確保手機與電腦連線至 **同一個 Wi-Fi**。
2. 使用手機掃描看板左上角的 QR Code。
3. 若無法開啟，請檢查 Windows 防火牆是否允許 **5000 埠 (Port)** 的通訊。

## 項目修改功能說明

現在具備「新增權限」的使用者可以直接修改已存在的項目內容與備註。

### 1. 編輯入口
在 **「進度管理」** 頁面中，項目的內容旁邊會出現一個灰色的 **「修改」** 按鈕。

### 2. 修改介面
點擊後會進入與新增頁面相同的表單，但會自動載入該項目的原始資料。修改完畢後按下 **「確認修改」** 即可套用。

### 3. 即時同步
修改後的內容會立即反映在管理頁面，且 **電視看板 (`/show`) 也會同步更新**。
