# POR 系統：兩岸跨境直連實施指南 (簡易 A 方案)

本指南旨在幫助非技術背景的大陸用戶，能以最簡單的方式（瀏覽器輸入網址）存取位於台灣的 POR 系統。

## 1. 台灣主機端設定 (電腦 A)

### 設定內網固定 IP (重要)
為了防止電腦重啟後內網 IP 變動導致路由器轉發失敗，請務必：
- **優先建議**：在路由器後台將 A 電腦的 **MAC 地址與 IP 綁定** (Static DHCP)。
- **手動設定**：或在 A 電腦的網路設定中直接指定一個固定的內網 IP (如 `192.168.1.100`)。

### Windows 防火牆開放
1. **控制台** > **系統與安全性** > **Windows 防火牆**。
2. 點擊左側 **進階設定**。
3. **輸入規則** (Inbound Rules) > **新增規則**。
4. 選擇 **連接埠** (Port) > **TCP** > 特定本機連接埠輸入 `5000`。
5. 選擇 **允許連線** 並命名為 `POR_Server`。

## 2. 路由器 (Router) 設定
1. 登入路由器管理頁面 (通常為 `192.168.1.1`)。
2. 尋找 **Port Forwarding (通訊埠轉發)** 或 **虛擬伺服器**。
3. 新增轉送規則：
   - **外部連接埠**：建議設為 `5566` (可自訂)。
   - **內部連接埠**：`5000`。
   - **內部 IP**：電腦 A 的 IP (例如 `192.168.1.100`)。
   - **通訊協定**：TCP。

## 3. 大陸端存取方式
在大陸電腦/平板的瀏覽器直接輸入：
`http://<您的台灣固定IP>:5566`

> [!TIP]
> 建議將此網址存為瀏覽器書籤，方便工廠人員一鍵開啟。

## 4. 安全與維護
- **修改密碼**：請務必修改 POR 系統 `admin` 帳號的預設密碼。
- **網路偵測**：若大陸端提示「無法連線」，請檢查台灣端的固定 IP 是否變動。
- **效能優化**：若操作卡頓，建議切換至「方案 B：資料庫鏡像同步」架構。
