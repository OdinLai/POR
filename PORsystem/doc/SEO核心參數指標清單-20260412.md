# SEO 核心技術參數指標清單

在架設 POR 行銷網站時，影響搜尋引擎排名 (SEO) 的主要參數可以分為以下三大維度：

---

## 1. 鏈上參數 (On-Page SEO) —— 搜尋引擎對內容的解讀
這些是網頁原始碼中可以直接控制的標籤與內容指標：

| 參數名稱 | 建議規格 / 作用 |
| :--- | :--- |
| **Title Tag (標題標籤)** | 50-60 字元。需內含核心關鍵字（如：智慧生產看板）。 |
| **Meta Description** | 120-150 字元。吸引點擊的摘要，影響點擊率 (CTR)。 |
| **H1 - H6 (標題層級)** | H1 必須唯一且包含核心關鍵字；H2-H6 建立內容層級。 |
| **URL Structure** | 語意化網址（如 `/manufacturing-signboard` 而非 `/p=123`）。 |
| **Alt Text (替代文字)** | 搜尋引擎看不懂圖片，必須透過 Alt 描述圖片內容。 |
| **Internal Links** | 內部連結密度與錨點文字 (Anchor Text) 的相關性。 |

---

## 2. 技術參數 (Technical SEO) —— 網站的健康度與速度
這部分主要影響搜尋引擎爬蟲的抓取效率與使用者體驗：

*   **Core Web Vitals (核心網頁指標)**：
    *   **LCP (最大內容渲染時程)**：建議 < 2.5 秒（載入速度）。
    *   **FID (首次輸入延遲)**：建議 < 100 毫秒（互動反應）。
    *   **CLS (累積版面配置偏移)**：建議 < 0.1（頁面穩定度）。
*   **Mobile-Friendliness**：行動裝置適應性（Responsive Design）。
*   **HTTPS**：安全性協議，目前是排名的基本門檻。
*   **Robot.txt & Sitemap.xml**：引導爬蟲抓取正確的頁面。
*   **Canonical Tag**：處理重複內容，告訴搜尋引擎哪一個是「正本」。

---

## 3. 權重與環境參數 (Off-Page & Content Quality)
*   **Domain Authority (網域權威度)**：外部連結 (Backlinks) 的質量與數量。
*   **E-E-A-T**：專業性、經驗、權威性、信任度（網站是否有聯絡資訊、專業背景描述）。
*   **Dwell Time (停留時間)**：使用者進入後是否停留閱讀，還是立刻跳出。
*   **Content Freshness**：內容更新頻率。

---

### **針對 POR 行銷網頁的特別建議：**
由於您計畫使用 **Cloudflare Pages**，在「技術參數」上已經具備極大優勢（極速、內建 HTTPS、全球邊緣節點）。
您目前最需要關注的是 **「鏈上參數」中的關鍵字佈局**。
