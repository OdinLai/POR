# 投影看板系統：啟動與功能說明

## 🚀 啟動方式
本系統已打包為 Docker 容器，請直接在資料夾根目錄執行批次檔：

1. **啟動系統**：雙擊 `🚀點我啟動看板.bat`
   - 首次啟動會下載環境，請稍候。
   - 啟動成功後，終端機顯示 `Accessing at http://localhost:5000` 即可關閉視窗（背景運行中）。
2. **停止系統**：雙擊 `🛑點我停止看板.bat`
   - 當不需要使用或要釋放記憶體時執行。

---

## 🔗 功能網址入口
系統預設運行於 `http://localhost:5000`

| 功能名稱 | 網址位置 | 說明 |
| :--- | :--- | :--- |
| **📺 電視看板** | [點我開啟](http://localhost:5000/show) | **主要展示頁**（建議電視用此網址，30秒自動刷新） |
| **➕ 新增訂單** | [點我開啟](http://localhost:5000/settings) | 用於點單、輸入內容與**備註** |
| **⚙️ 進度管理** | [點我開啟](http://localhost:5000/manage) | 控制項目轉移（到貨、生產等）與**修改看板標題** |
| **👤 用戶管理** | [點我開啟](http://localhost:5000/users) | 新增員工帳號、分配讀寫權限 (僅限管理員) |
| **📊 歷史紀錄** | [點我開啟](http://localhost:5000/all_data) | 查看已清除的歷史資料與時程 |

---
